# Finance Dashboard Monorepo

이 저장소에는 study-todo라는 Vite + React 샘플 앱이 포함되어 있습니다.

실행 방법

1. study-todo 디렉터리로 이동:
   cd study-todo
2. 의존성 설치:
   npm install
3. 개발 서버 시작:
   npm run dev
   (기본: http://localhost:5173)

빌드 및 미리보기

1. 빌드:
   npm run build
2. 빌드 결과 미리보기:
   npm run preview
